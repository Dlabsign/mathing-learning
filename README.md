# 🎮 MathingLearning

MathingLearning adalah platform pembelajaran matematika interaktif berbasis web yang dirancang khusus untuk anak-anak. Website ini menggabungkan konsep belajar dan bermain (edugame) agar proses memahami matematika menjadi lebih menyenangkan, tidak membosankan, dan lebih mudah dipahami.

---

## 📌 1. Fungsi Website

Website ini dibuat dengan tujuan utama:
- Membantu anak-anak belajar matematika dengan cara yang menyenangkan
- Mengurangi rasa takut atau bosan terhadap pelajaran matematika
- Melatih kecepatan dan ketepatan dalam berhitung
- Memberikan pengalaman belajar interaktif melalui game
- Mengasah kemampuan logika dan problem solving sejak dini

Fitur utama:
- Game matematika (penjumlahan, pengurangan, perkalian, dll)
- Sistem poin dan tantangan
- Timer yang bisa disesuaikan
- Statistik progres pengguna

---

## 🚀 2. Cara Menggunakan Website

Berikut langkah-langkah penggunaan:

1. **Buka Website**
   - Akses halaman utama (Landing Page)

2. **Daftar / Login**
   - Klik tombol *Daftar* untuk membuat akun baru
   - Atau klik *Login* jika sudah memiliki akun

3. **Pilih Menu Permainan**
   - Masuk ke halaman menu
   - Pilih jenis permainan matematika yang ingin dimainkan

4. **Atur Timer (Opsional)**
   - Tentukan batas waktu per soal (misalnya 5 detik, 10 detik, dll)

5. **Mulai Bermain**
   - Jawab soal yang muncul
   - Kumpulkan poin dan tingkatkan skill

6. **Lihat Hasil**
   - Setelah selesai, lihat skor dan progres belajar

## 🛠️ 3. Teknologi yang Digunakan

Website ini dibangun menggunakan teknologi modern:

### Frontend
- React.js
- JavaScript (ES6+)
- CSS (Custom Styling, Claymorphism UI)

### Routing & State
- React Router
- React Hooks (useState, useEffect)

### Backend (opsional jika digunakan)
- Firebase Authentication (untuk login/register)
- Firebase / API (untuk penyimpanan data)
